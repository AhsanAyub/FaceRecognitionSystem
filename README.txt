===== FACE DETECTION SYSTEM =====

1. 	This is built bearing the concept of security guards/ watchmans of dhaka
	city.

2.	Only face's picture saving is done from live feeding via webcam done till then.

3.	OpenCV 2.4.1 is used to create this project and is built in JAVA environment.


***** Flow of actions *****

1.	Capture screen from capture button displayed besides live feeding.
2.	Temporary saved picture will then be prompted to save as a .png file.
3.	Even if all the initials/ insertion fields are empty, picture will be saved
	with system time as name.
